## GitHub

Anlegen eines neuen Repositories, indem man ein `README.md` erstellt. Diese auch auf `public` stellen.

Auf Kali erstellt man dann einen neuen Ordner `mkdir SYTB` und in diesen Ordner kopiert man dann das Repository mit dem Link von der Seite auf Kali Linux mithilfe dieser Zeile:
```
┌──(kali㉿kali)-[~/SYTB/3AHITS-SYTB-Petereder-Leah]
└─$ git clone https://github.com/leahsophiep/3AHITS-SYTB-Petereder-Leah
```

## VsCode installieren

Download 

```sh
┌──(kali㉿kali)-[~/Downloads]
└─$ sudo apt install ./code_1.109.5-1771531656_amd64.deb
```

und dann um Visual Code zu starten muss man `code` eingeben.

Dann im VsCode die Zeilen:
```
git config --global user.name "Leah Petereder"
```

und

```
git config --global user.email leah.petereder@htl-braunau.at
```

eingeben.

Danach auf VsCode links im Bildschirm auf dieses Zeichen klicken:

![alt text](image.png)

Daraufhin eine `Message` eingeben und danach auf `Commit` und dann `Sync Changes` klicken.

Jetzt sieht man, dass es nach Github kopiert wurde.
