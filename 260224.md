```

└─$ git clone https://github.com/leahsophiep/3AHITS-SYTB-Petereder-Leah
```

## 2. VsCode installieren

Erstmal im Browser im Kali Linux `VsCode Linux` eingeben und dann den Link, welcher dasteht kopieren und damit diese Zeile schreiben:

```sh
┌──(kali㉿kali)-[~/Downloads]
└─$ sudo apt install ./code_1.109.5-1771531656_amd64.deb
```

und dann um Visual Code zu starten muss man `code` eingeben.

Dann im VsCode im Command prompt die Zeilen:
```
git config --global user.name "Leah Petereder"
```

und

```
git config --global user.email leah.petereder@htl-braunau.at
```

eingeben.

Danach auf VsCode links im Bildschirm auf dieses Zeichen klicken:

![alt text](image.png)

Daraufhin eine `Message` eingeben und danach auf `Commit` und dann `Sync Changes` klicken.

Jetzt sieht man, dass es nach Github kopiert wurde und kopiert den Namen des geänderten files und klickt dann in `Settings` dann `Pages` und daraufhin beim Link auf `Visit Site` und am Ende fügt man den Namen des files an und dann `.html`
